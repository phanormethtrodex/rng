```
Usage: rng [bondxluazcys] [#]

# - number of characters

[bondxluazcys] - character class:
b - binary
o - octal
n - "nonal" (1-9)
d - decimal
x - hexadecimal
l - lower
u - upper
a - alpha
z - alphanumeric
c - "printable" ascii
y - non-alphanumeric "printable" ascii
s - any ascii
no args - random number of chars from random class
```
